Design Guidelines

Calm Futurism for agentic products: mono palette, liquid glass depth, predictable structure, and disciplined readability. This page is the single source of truth for UI decisions across ensemble, playground, and future modules.

Monochrome Liquid Roboto Body Special Gothic Display 8px Spacing Grid

Core Principles

Make every screen feel quiet, precise, and operational. Visual beauty serves decision speed, not decoration.

P1

Hierarchy First

One dominant focal action per viewport. Secondary actions become chips, tertiary actions become text links.

P2

Contrast by Layers

Use density and blur depth before adding color. Keep semantic colors only for critical status and alerts.

P3

Rhythm over Noise

Maintain strict spacing cadence. Prefer repeated panel templates to reduce visual switching cost.

P4

Readable Always

Minimum 14px body text, stable line-height, and compressed but scannable rows for high-density dashboards.

Design Tokens

Tokens are platform-neutral. Keep names and values synced across CSS variables, docs, and component defaults.

Accessibility & QA

Beautiful systems still fail without readability, keyboard safety, and consistent behavior under stress.

WCAG-aligned Checks

Must Pass Before Shipping

  • Touch targets≥ 44 × 44
  • Keyboard trapZero allowed
